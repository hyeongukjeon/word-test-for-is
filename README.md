# Word Test for IS

A mobile-first English vocabulary quiz site with mock tests, High School 250 words, TEPS vocabulary, and local bookmarks.

## Publish Bundle

GitHub Pages only needs these runtime files:

- `index.html`
- `styles.css`
- `app.js`
- `teps-data.js`

Large source PDFs, original OCR images, and local extraction backups are intentionally ignored by Git.
